关于组件：
  1、引用组件
      import Header from './header'
      import Footer from './footer'
  2、注册组件
      new Vue({
        data:{},
        components:[Header,Footer]
      })
  3、 组件之间通信之父组件向子组件传值   props
      new Vue({
          data:{},
          props:['msg'],
          methods:{
              doThis:function(){
                  console.log(this.msg)
              }
          }
      })
      <!--  this is app.vue  -->
      <header msg='something'></header>

  4、组件之间通信之子组件向父组件传值   v-on:父组件监听事件。 $emit():子组件触发事件
